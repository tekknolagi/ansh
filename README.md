Ansh
====

Ansh is a Great Intern. As such, I have made this humble Node.js module in his honor.

## Installation

    npm install ansh --save

## Usage

    var ansh = require('ansh');
    console.log(ansh.ansh());

OR

    $ ansh

## Tests

    npm test
